# CONTRIBUTING

Thank you for considering contributing to the MinID-X-XRDXRF project! We appreciate your efforts to improve our project.

## Contribution Guidelines
- Ensure your contributions align with the project goals.
- Discuss major changes with the team before making a pull request.

## Development Setup
### Android
1. Clone the repository:
   ```bash
   git clone https://github.com/smainechellat-star/MinID-X-XRDXRF.git
   ```
2. Open the project in Android Studio.
3. Install the required SDKs and dependencies through the Android Studio setup wizard.

### Python
1. Clone the repository:
   ```bash
   git clone https://github.com/smainechellat-star/MinID-X-XRDXRF.git
   ```
2. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
3. Activate the virtual environment:
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```
   - On macOS/Linux:
     ```bash
     source venv/bin/activate
     ```
4. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Bug Reporting Procedures
1. Check the existing issues to see if it’s already reported.
2. If not, create a new issue with detailed information:
   - **Title**: A brief summary of the issue
   - **Description**: Detailed explanation of the bug
   - **Steps to Reproduce**: Clear steps to replicate the issue
   - **Expected vs Actual Outcome**: What you expected and what happened

## Code Style Guidelines
- Follow PEP 8 for Python code.
- Use meaningful variable names and comments.
- Adhere to the project’s coding standards as much as possible.
